# CoursAI — дашборд управления ML-обучением

Десктоп-приложение (Electron + React + Python/FastAPI), которое наглядно показывает
«где я сейчас и над чем работать» в процессе ML-обучения. Прогресс отслеживается по трём
измерениям: **Теория → Практика → Проекты**.

Источник структуры — Markdown-документы в `docs/` (курс, трекер практики, 90-дневный
backlog, RAG MVP, Definition of Done). Документы **read-only** — приложение их не меняет.
Прогресс хранится отдельно в SQLite (в пользовательской папке), чтобы исходные `.md` не
модифицировались.

## Требования

- **Node.js** v24, **npm** v11
- **Python** 3.13
- **uv** (менеджер Python-зависимостей)
- ОС: Windows 11 (разработка велась на PowerShell; пути/команды учитывают Windows)

## Установка

```powershell
# 1. Зависимости фронтенда (Electron, React, Vite)
npm install

# 2. Виртуальное окружение и зависимости бэкенда
cd backend
uv venv
uv pip install -r requirements.txt
cd ..
```

## Запуск

**Простой способ — двойной клик по `start.bat`** в корне проекта. Скрипт сам проверит
venv и `node_modules`, при необходимости доустановит и поднимет приложение.

**Без чёрного окна консоли — двойной клик по `CoursAI.vbs`.** Это та же логика, что и
`start.bat`, но окно cmd скрыто; открывается только окно приложения. Если запуск
почему-то не удался, запусти `start.bat` напрямую — там будут видны сообщения об ошибке.

**Вручную (dev-режим):**

```powershell
npm run dev
```

Electron-окно открывается сразу; Python-бэкенд (FastAPI) поднимается параллельно на
свободном порту `127.0.0.1` как дочерний процесс и гасится при закрытии окна. В шапке
главного экрана отображается статус связи с бэкендом (**Backend: ok**).

> Первый холодный запуск может занять до ~минуты (антивирус сканирует свежий venv) — это
> нормально, окно дождётся готовности бэкенда.

## Навигация

Левое меню — пять разделов:

- **Главная** — обзор: измерения прогресса и текущий фокус «над чем работать».
- **Курс** — 10 модулей курса: цель, что изучить, задание, критерий зачёта, статус.
- **Практика** — трекер практики и журнал артефактов.
- **90 дней** — backlog по фазам и неделям с чек-листами и метриками.
- **RAG MVP** — критерии приёмки MVP и dev-backlog по версиям.

## Структура проекта

```
app/                      Electron + React + TypeScript (фронтенд)
  src/main/               main-процесс Electron (окно, запуск бэкенда)
  src/preload/            preload-мост (contextBridge)
  src/renderer/           React-приложение (экраны, роутинг, API-клиент)
backend/                  Python + FastAPI
  app/main.py             точка входа, /health, CLI --host/--port
  app/parsing/            парсинг Markdown в сущности (курс, практика, backlog, MVP, DoD)
  tests/                  тесты (pytest)
docs/                     исходные Markdown-документы (read-only источник структуры)
docs/tasks/               план реализации (бэклог эпиков E1–E7) и промт его генерации
start.bat                 лаунчер «в один клик»
```

## Разработка

```powershell
npm run build       # production-сборка (out/)
npm run typecheck   # проверка типов TypeScript

# Тесты бэкенда
cd backend
.venv/Scripts/python.exe -m pytest -q
```

> При выводе кириллицы из Python в консоль Windows используйте
> `PYTHONIOENCODING=utf-8`, иначе возможен `UnicodeEncodeError` (cp1251).
