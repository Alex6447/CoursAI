Этот проект создан для управления процессом обучения и создания кодовых отчетных заданий. Управление осуществляется путем взаимодействия с дашбордом наглядным визуальным UI компонентом в котором сразу видно где я нахожусь и над чем сейчас нужно работать и как продвигается прогресс в обучении по всем заданным критериям Теория - Практика - Создание проектов и так далее. Дашборд реализован в Electron с применением самых современных технологий бек на python а UI на твое усмотрение но с современным дизайном и технологиями.
